# WebChatting
의존성없는 채팅서비스

현재 React, View.js, Angular JS 등등 수많은 JS 프레임워크들이 나오고 있습니다.
계속해서 나오는 프레임워크들을 익히고 공부하는 것이 필수적이나 그전에 가장 먼저 근본을 공부하는 것이 우선이라 생각하였습니다.
따라서 vanilla JS만으로 의존성 없는 채팅서비스를 기획했습니다.
